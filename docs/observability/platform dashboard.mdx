# Platform Dashboard

Head to the `Platform Report` tab from `Insights` panel for a project's Console, and you'll see the new platform metrics.
This dashboard, offers a visual representation of key metrics across four main categories that help with an overall assessment of the Data Delivery Network platform:

## Security and Governance (DDN Free)

- Public Endpoint Alert: Quickly identify any models exposed without authentication.
- Security Issue Resolution: Step-by-step guide on how to fix detected security issues.
